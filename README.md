# Curate Your Life (Next.js)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize
- Replace the `mailto:` in `app/page.tsx` with your Instagram/WhatsApp DM link.
- Update `metadataBase` in `app/layout.tsx` to your real domain.
- Swap images in `public/images/` if needed.
